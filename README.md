# ThunderbirdUpdater

Single-file WinForms application to automatically download and update Thunderbird Portable.

## How to use

1. Place `7zr.exe` (https://www.7-zip.org/a/7zr.exe) in the same folder as `ThunderbirdUpdater.exe`.
2. Run `ThunderbirdUpdater.exe`.
3. Click "Update Thunderbird Portable".
4. Logs are stored in `update.log` and `version.log` in the application folder.

## Requirements

- Windows
- .NET 10 runtime (or self-contained build)
